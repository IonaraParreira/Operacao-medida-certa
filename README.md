#Acesso...#

Você está tentando se proteger de uma doença que atingiu proporções de epidemia,tornando-se um dos maiores desafios de saúde pública do século XXI.Hoje mais de 1 milhão de pessoas no mundo vivem com ela.

Felizmente você pode se proteger e escapar.
A chave para isso é entrar em um bunker, que tem uma senha.Mas não se preocupe, você está diante de um sistema inteligente que foi elaborado para permitir que apenas os que não tem a doença possam sobreviver.


Como no mundo nem tudo são flores,o sistema não é para os fracos, é apenas para aqueles que depois de verem o resultado tem 2 escolhas.

Fazer algo ou se recusar a Fazer.
O que você fará?
Você decide!
